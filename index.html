Sweepstakes Registration

During the migration of a sweepstakes funnel from an HTML/vanilla JavaScript platform to a React platform I built original React components as well as adapted existing components from our alternate registration funnels. This often involved working closely with management as to the requirements for the various elements of the site as well as colaborating with other developers working on related pieces.

For this assignment I implemented a theming methodology I created for making our registration funnel components reusable and new funnels quickly themable. This structure modularized our CSS into separate levels using Less preprocessing to allow specific targeting where needed. Implementation of this architecture sped further initiatives and split testing for the funnel by 3x.

Bankrate Loader

For this project I improved the UX for and rebuilt the introductory section for the Bankrate Credit Check Funnel using React, closely coordinating work with another teammember overseeing styling. The implementation involved sending user data to the server, awaiting a valid response, and displaying the call-to-action. By slowing the speed of the loader I insured the server had time to verify user data and the call-to-action could immediately be displayed. By cleaning up the previous implementation of state I improved code maintainability and prevented future bugs.
